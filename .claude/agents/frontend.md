# Frontend Agent

## Responsibilities

- UI components
- Layouts
- Dashboards
- Responsive design

## Tech Stack

- Next.js
- React
- TailwindCSS
- shadcn/ui

## UI Principles

The UI should be:

- Minimal
- Fast
- Keyboard friendly
- Distraction free

Design inspiration:
- Notion
- Linear
- Raycast

## Key Features

- Global command palette (Ctrl+K) for searching notes, tasks, commands, and pages
- Dashboard displaying: today's tasks, habit progress, stock watchlist, tech news, quick notes

## Guidelines

- Use TypeScript
- Follow Next.js best practices
- Generate clean and readable code
- Use reusable components
- Separate UI and business logic
- Prioritize maintainability
