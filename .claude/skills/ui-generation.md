# Skill: UI Generation

Generate Tailwind + React components.

## Stack

- React (with Next.js)
- TailwindCSS
- shadcn/ui

## UI Principles

- Minimal and distraction-free
- Fast and responsive
- Keyboard friendly (support Ctrl+K command palette)
- Inspired by Notion, Linear, and Raycast

## Guidelines

- Use shadcn/ui as the base component library
- Style with TailwindCSS utility classes
- Keep components small and focused (single responsibility)
- Separate presentational components from logic
- Support dark mode where applicable
